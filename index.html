IPS by ashankar

IPS (IGNOU Prashnottari Sewa) is Query Management System developed by ACIIL, IGNOU under the guidance of Prof. K. R. Srivathsan, Pro-Vice Chancellor of IGNOU and implemented in IGNOU. IPS is released in Free and Open Source Software Distribution under LGPL license.

IPS is a Query Management System. Its provide customizable interfaces to the User who can send queries to the concern department of IGNOU and get back response from the IGNOU.

Dependencies

JDK1.6 or above, Application Server, NetBeans6.8 or above

Install

IPS is a web application developed using Java, Servlet and JSP. Download the IPS code which is in NetBeans structure and open it in NetBeans6.8 or above version.
